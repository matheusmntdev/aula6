Guia de Comandos MySQL via CLI

1. Acessando o MySQL via CLI

Para acessar o MySQL via terminal, use o comando: 

mysql -u [usuário] -p

Substitua [usuário] pelo nome do usuário do MySQL. O comando solicitará a senha do usuário.

Exemplo: 

mysql -u root -p

2. Comandos Básicos do MySQL

a) Mostrar Bancos de Dados Existentes

SHOW DATABASES;

b) Criar um Novo Banco de Dados

CREATE DATABASE nome_do_banco;

Exemplo: 

CREATE DATABASE meubanco;

c) Selecionar um Banco de Dados

USE nome_do_banco;

Exemplo: 

USE meubanco;

d) Mostrar Tabelas do Banco de Dados

SHOW TABLES;

e) Descrever uma Tabela

DESCRIBE nome_da_tabela;

Exemplo: 

DESCRIBE clientes;

3. Comandos para Manipulação de Dados

a) Criar uma Tabela

CREATE TABLE nome_da_tabela (
    coluna1 tipo_de_dado restrições,
    coluna2 tipo_de_dado restrições,
    ...
);

Exemplo: 

CREATE TABLE clientes (
    id INT PRIMARY KEY AUTO_INCREMENT,
    nome VARCHAR(100) NOT NULL,
    email VARCHAR(100) UNIQUE,
    data_nascimento DATE
);

b) Inserir Dados em uma Tabela

INSERT INTO nome_da_tabela (coluna1, coluna2, ...)
VALUES (valor1, valor2, ...);

Exemplo: 

INSERT INTO clientes (nome, email, data_nascimento)
VALUES ('João Silva', 'joao.silva@example.com', '1990-05-15');

c) Consultar Dados

SELECT coluna1, coluna2, ...
FROM nome_da_tabela
WHERE condição;

Exemplo: 

SELECT nome, email FROM clientes WHERE data_nascimento > '1985-01-01';

d) Atualizar Dados

UPDATE nome_da_tabela
SET coluna1 = valor1, coluna2 = valor2, ...
WHERE condição;

Exemplo: 

UPDATE clientes SET email = 'joao.novo@example.com' WHERE id = 1;

e) Excluir Dados

DELETE FROM nome_da_tabela
WHERE condição;

Exemplo: 

DELETE FROM clientes WHERE id = 1;

4. Comandos para Gerenciamento de Estrutura

a) Adicionar uma Coluna a uma Tabela

ALTER TABLE nome_da_tabela
ADD COLUMN nome_da_coluna tipo_de_dado restrições;

Exemplo: 

ALTER TABLE clientes
ADD COLUMN telefone VARCHAR(15);

b) Modificar uma Coluna

ALTER TABLE nome_da_tabela
MODIFY COLUMN nome_da_coluna novo_tipo_de_dado;

Exemplo: 

ALTER TABLE clientes
MODIFY COLUMN telefone VARCHAR(20);

c) Excluir uma Coluna

ALTER TABLE nome_da_tabela
DROP COLUMN nome_da_coluna;

Exemplo: 

ALTER TABLE clientes
DROP COLUMN telefone;

d) Excluir uma Tabela

DROP TABLE nome_da_tabela;

Exemplo: 

DROP TABLE clientes;

e) Excluir um Banco de Dados

DROP DATABASE nome_do_banco;

Exemplo: 

DROP DATABASE meubanco;

5. Comandos para Gerenciamento de Usuários e Permissões

a) Criar um Novo Usuário

CREATE USER 'nome_do_usuario'@'localhost' IDENTIFIED BY 'senha';

Exemplo: 

CREATE USER 'novousuario'@'localhost' IDENTIFIED BY 'minhasenha';

b) Conceder Permissões a um Usuário

GRANT tipo_de_permissão ON nome_do_banco.nome_da_tabela TO 'nome_do_usuario'@'localhost';

Exemplo: 

GRANT SELECT, INSERT, UPDATE ON meubanco.* TO 'novousuario'@'localhost';

c) Revogar Permissões

REVOKE tipo_de_permissão ON nome_do_banco.nome_da_tabela FROM 'nome_do_usuario'@'localhost';

Exemplo: 

REVOKE INSERT ON meubanco.* FROM 'novousuario'@'localhost';

d) Excluir um Usuário

DROP USER 'nome_do_usuario'@'localhost';

Exemplo: 

DROP USER 'novousuario'@'localhost';

6. Comandos para Sair e Outros Utilitários

a) Sair do MySQL

EXIT;

ou 

QUIT;

b) Mostrar Versão do MySQL

SELECT VERSION();

c) Mostrar Status do Servidor

STATUS;

7. Importar e Exportar Bancos de Dados

a) Exportar um Banco de Dados

mysqldump -u [usuário] -p nome_do_banco > arquivo.sql

Exemplo: 

mysqldump -u root -p meubanco > backup.sql

b) Importar um Banco de Dados

mysql -u [usuário] -p nome_do_banco < arquivo.sql

Exemplo: 

mysql -u root -p meubanco < backup.sql

Conclusão

Este guia cobre os comandos essenciais para trabalhar com o MySQL via CLI. Com esses comandos, você pode gerenciar bancos de dados, tabelas, usuários e permissões, além de realizar operações de manipulação de dados. Pratique esses comandos para se familiarizar com o MySQL e aumentar sua produtividade no gerenciamento de bancos de dados.
